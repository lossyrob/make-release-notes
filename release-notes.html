GeoTrellis 0.8.2 is now available!

A new minor release of GeoTrellis, 0.8.2, which is now available as a release candidate: 0.8.2-RC2. Please let us know if you find any issues with the release candidate so we can address them before the release of 0.8.1.

There are some exciting new features in this release:

Geotrellis Transit

We've included new functionality for creating shortest path trees and creating transit shed rasters using the new geotrellis.network package. Please see transit.geotrellis.com for more info, including demos and an API listing.
